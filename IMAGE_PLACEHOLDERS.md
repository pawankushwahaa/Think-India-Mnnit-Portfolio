## Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 
  - Event images: 800x600px or larger
  - Gallery images: 1200x800px or larger
  - Team photos: 400x400px (square) or 400x500px (portrait)
  - Logo: 200x200px or larger
- **File Size**: Keep images optimized (under 500KB each for web performance)

## Adding Images

1. Simply place your images in the corresponding folders with the exact filenames listed above
2. The website will automatically display them
3. If an image is missing, a broken image icon will appear - replace it with the correct image

## Note

All image paths are relative to the `public` folder. The website references them as `/events/...`, `/gallery/...`, etc.
