1. Save your Think India logo image as `think-india-logo.png`
2. Save your MNNIT logo image as `mnnit-logo.png`
3. Place both files in the `frontend/public/` folder
4. The website will automatically display them

## File Format

- **Background**: Transparent background recommended for Think India logo
- **Quality**: High resolution for best display
